## API Endpoints

### Необхідні для роботи
```javascript
// 1. Отримання конфігурації
GET /config
Response: { dbType: 'sybase', hasAnalysisRegistration: true }

// 2. Пошук пацієнтів
POST /search
Body: { lastName, firstName, page, limit }
Response: { results: [...], total: number }

// 3. Завантаження аналізів
POST /api/analyses
Response: [{ nazov, kodvys, cena }, ...]

// 4. Реєстрація аналізів (НОВИЙ)
POST /api/register-analyses
Body: { patient, analyses, notes, registrationDate }
Response: { success: true, id: "..." }
```

## UX Поліпшення

### 🎯 Чіткість
- **Раніше**: Користувач не знав, де що шукати
- **Тепер**: Кожен крок має ясну назву та описання

### 📍 Орієнтація
- **Раніше**: Немає показника прогресу
- **Тепер**: Step indicator показує де ти і куди йти

### 🔄 Навігація
- **Раніше**: Складна логіка для повернення назад
- **Тепер**: Зручні кнопки "← Назад" на кожному блоці

### ✓ Валідація
- **Раніше**: Помилки при клікові на кнопку
- **Тепер**: Перевірка перед переходом на наступний блок

### 👁️ Перегляд
- **Раніше**: Без можливості перевірити дані перед реєстрацією
- **Тепер**: Окремий блок для перегляду всього

## Як користувач взаємодіє з формою

### Сценарій: Реєстрація аналізу для пацієнта

```
1. На сторінці з'являється Блок 1 (Пошук пацієнта)
2. Користувач вводить прізвище "Іванов"
3. Натискає "Пошук"
4. Видно 10 результатів + пагінація
5. Користувач вибирає "Іванов Іван" (перший результат)
6. Форма автоматично переміщується на Блок 2
7. Показується Step Indicator: ①✓ ─ ② ─ ③
8. Користувач видить список всіх доступних аналізів
9. Натискає на "Загальний аналіз крові"
10. Аналіз додається в панель "Обрані"
11. Натискає на "Біохімія крові"
12. Й ще один аналіз додається
13. Сума оновлюється: 370.50 грн
14. Натискає "Перейти до підтвердження"
15. Форма переміщується на Блок 3
16. Step Indicator: ①✓ ─ ②✓ ─ ③
17. Показується:
    - Дані пацієнта: Іванов Іван Іванович...
    - Обрані аналізи з цінами
    - Загальна сума
18. Користувач опціонально додає примітку
19. Натискає "Зареєструвати аналізи"
20. Показується Success Modal: "Успішно!"
21. Через 2 секунди форма скидається до Блоку 1
22. Готово!
```
